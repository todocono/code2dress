NodeVisitor<SoyNode,Void>
public class RenderVisitor
extends AbstractSoyNodeVisitor<Void>

Visitor for rendering the template subtree rooted at a given SoyNode.

Important: Do not use outside of Soy code (treat as superpackage-private).

The rendered output will be appended to the Appendable provided to the constructor.
